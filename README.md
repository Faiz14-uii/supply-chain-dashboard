# Supply Chain Analysis Dashboard

Dashboard interaktif untuk analisis supply chain menggunakan Streamlit.

## Features

- Real-time monitoring dengan auto-refresh
- Analisis regresi 6 metode
- Clustering supplier dengan K-Means
- Time series forecasting ARIMA
- Visualisasi interaktif dengan Plotly

## Dataset

Dataset mencakup informasi supply chain dengan fitur:

- Informasi produk dan supplier
- Data penjualan dan profit
- Analisis clustering supplier
- Forecast time series

## Technology Stack

- Python 3.11+
- Streamlit 1.48.0
- Pandas, Plotly, Scikit-learn
- ARIMA Time Series Forecasting
